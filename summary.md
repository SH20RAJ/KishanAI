## ğŸ—ï¸ Architecture Overview
The system follows a **Microservices-based Serverless Architecture**:

1.  **User Interface:**
    -   **Telegram Bot:** The primary interface for farmers (low barrier to entry).

2.  **API Layer (`src/app/api`):**
    -   **`/telegram`:** Handles webhooks, routing user messages to specific agents (Chat, Disease, Weather).
    -   **`/application`:** Serves the web frontend needs (Marketplace, Profile, Services).

3.  **AI Engine:**
    -   Processes images and text inputs.
    -   Retrieves context from the database.
    -   Generates responses in the user's native language.
