```

[optional body]

[optional footer(s)]
```

- `chore`: Maintenance tasks
- `perf`: Performance improvements
- `ci`: CI/CD changes

#### **Examples:**
```bash
feat(api): add crop recommendation endpoint
fix(ui): resolve mobile layout issues on disease detection page
docs(readme): update installation instructions
```
